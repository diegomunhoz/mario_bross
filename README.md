Jogos desenvolvido no curso de Pygame Avançado - Q-CURSOS NETWORK: neste curso é criado o jogo do Super Mário Bross, um projeto muito grande com diversas classes e centenas de funções.

- Criando estrutura de pastas
- Criando arquivos necessários
- Iniciando com o arquivo Setup
- Configurando as teclas 
- Criando a classe de inicialização
- Criando as constantes
- Configurando a tela inicial
- Iniciando com a tela de fundo
- Configurando classe principal
- Classe principal do jogo
- Caminho das pastas
- Passando a referencia das pastas
- Criando o arquivo main principal
- Classe de Load_screen
- Arquivo de informações
- Sons e músicas
- Criando da classe do menu
- Executando o jogo para testes
- Verificando itens posicionados
- Classe do nivel 1 do jogo
- Fundo do cenário e objetos
- Colisões com os objetos
- Classe dos tijolos
- Classe dos inimigos
- Pontuação durante o jogo
- Iniciando os testes finais
- Finalizando o jogo
